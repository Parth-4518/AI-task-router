⚙️ How It Works

1. Keyword Matching

  • Role match: +10 points
  • Title match: +8 points
  • Name match: +6 points
  • Capability match: +3 points
  • Generic keyword: +4 points

2. Priority Detection

  • Critical: urgent, critical, emergency, down, broken
  • High: important, high priority, blocking
  • Medium: default
  • Low: nice to have, low priority

3. Issue Generation

  • Auto-generates title
  • Creates description
  • Sets priority
  • Sets status to backlog
